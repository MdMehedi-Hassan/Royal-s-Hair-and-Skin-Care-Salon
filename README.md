# Royal's – Hair & Skin Care Salon

A responsive and modern website for **Royal's Salon**, built with **HTML5** and **TailwindCSS**.  
The site showcases services, team members, founder details, and provides quick contact information.  

## 🚀 Features
- 🎨 **Responsive Design** – works seamlessly on desktop and mobile  
- 🖼️ **Hero Section** – bold introduction with call-to-action  
- 📖 **About & Founder Section** – story and philosophy of Royal’s Salon  
- 👨‍👩‍👦 **Team Section** – highlight of grooming experts  
- 💈 **Services Section** – categorized pricing tables for grooming & styling  
- 📍 **Contact Section** – phone, address, and map integration (optional)  
- 🌐 **SEO Optimized** – semantic tags and meta description  

## 🛠️ Tech Stack
- **HTML5**  
- **TailwindCSS**  
- **JavaScript (vanilla)** for mobile navigation toggle
